# Global Superstore Profitability Optimization Analysis

## 📌 Objective
To analyze profitability drivers and identify margin leakage areas using Pandas and NumPy.

---

## 📊 Dataset
Global Superstore Dataset (Kaggle)

Rows: 51,290  
Columns: 27  

---

## 🔍 Key Insights

- Overall Profit Margin: ~11.6%
- Furniture category margin: 6.9%
- Tables sub-category margin: -8.46% (Loss-making)

### Root Causes:
- Average Discount (Tables): 29%
- Other Furniture Discount: 15.6%
- Tables Shipping Cost: $92.75
- Other Furniture Shipping Cost: $39.98

High discount + high shipping cost = Margin erosion.

---

## 💡 Recommendations

- Cap discount at ≤20%
- Introduce bundle-based discounting
- Optimize logistics for bulky products
- Apply region-level pricing strategy
